# Шёпот Ниже

Небольшой автономный прототип веб-3D хоррор-бродилки: тёмный лабиринт, сбор страниц, преследующая сущность и один выход.

## Запуск

- Самый простой вариант: откройте `index.html` в современном браузере.
- Прототип использует `three.js` и `PointerLockControls` с CDN, поэтому для запуска нужен доступ к интернету.
- Если браузер ограничивает локальные модули или шрифты, поднимите простую статическую раздачу из этой папки и откройте локальный адрес.

## Управление

- `W`, `A`, `S`, `D` — ходьба
- `Shift` — ускорение
- `Space` — прыжок
- Мышь — обзор
- `F` — включить или выключить фонарик
- `B` — пауза и настройки чувствительности
- `E` — подобрать страницу или открыть выход
- `Esc` — освободить курсор

## Телефон

- На тач-устройствах игра сама включает мобильный режим.
- Слева появляется джойстик для движения.
- Справа работает свайп-зона для обзора камеры.
- Кнопки `Run`, `Jump`, `Use`, `Light`, `Pause` появляются автоматически только на телефоне.

## Цель

Соберите 4 шепчущие страницы, не потеряйте рассудок и доберитесь до двери с зелёной печатью.
